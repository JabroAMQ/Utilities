# Google Chrome & Windows Keyboard Shortcuts Reference

This document serves as a reference guide for `Ctrl + [Key]` combinations in Google Chrome (Windows/Linux).

It may contains some inaccuracies, and shortcuts might change depending on your specific browser version, Operating System or other userscript you have installed.

Therefore, it is highly advisable not to take this document as the absolute truth; always test the default behavior yourself before overriding any key combinations.

## :white_check_mark: Recommended Keys :white_check_mark:
These keys have no native assignment in Chrome or are useless for AMQ purposes. Ideal to use for your custom shortcuts.

| Shortcut | Chrome Native Action |
| :--- | :--- |
| `Ctrl + B` | Toggle Bold (Rich Text, Useless Inside AMQ) |
| `Ctrl + D` | Bookmark Current Page |
| `Ctrl + E` | None (legacy search focus) |
| `Ctrl + G` | Find Next Match |
| `Ctrl + H` | Open History Tab |
| `Ctrl + I` | None |
| `Ctrl + J` | Open Downloads Tab |
| `Ctrl + K` | Insert Link (Rich Text, Useless Inside AMQ) |
| `Ctrl + M` | None |
| `Ctrl + Q` :warning: | Close browser on Mac |
| `Ctrl + U` | View Page Source |

## :warning: Overridable, but not recommended :warning:
These keys are overridable but it might be interesting to keep their default usage for AMQ purposes. It is advisable not to override these.

`Ctrl + O`, `Ctrl + P` and `Ctrl + S` might produce some problems. Keep them as the last available options.

| Shortcut | Chrome Native Action |
| :--- | :--- |
| `Ctrl + A` | Select All Text |
| `Ctrl + C` | Copy Text |
| `Ctrl + F` | Open Find Bar |
| `Ctrl + O` :warning: | Open Local File Modal |
| `Ctrl + P` :warning: | Open Print Layout Window |
| `Ctrl + S` :warning: | Save Current Web Page |
| `Ctrl + V` | Paste Text |
| `Ctrl + X` | Cut Text |
| `Ctrl + Y` | Redo Typing |
| `Ctrl + Z` | Undo Typing |

## :no_entry_sign: Restricted Keys :no_entry_sign:
These keys are not overridable. Do not try to use them.

| Shortcut | Chrome Native Action |
| :--- | :--- |
| `Ctrl + L` | Focus Address Bar |
| `Ctrl + N` | Open New Window |
| `Ctrl + R` | Reload Page |
| `Ctrl + T` | Open New Tab |
| `Ctrl + W` | Close Tab |
